# 矩阵线上题目知识点

# 第一章、线性空间引论

## 一、 线性空间与子空间的基础概念

### 1. 映射类型

- **单射**：自变量不同，因变量必不同（$x_1 \neq x_2 \implies f(x_1) \neq f(x_2)$）。
- **满射**：值域等于目标集，即结果空间中的每个元素都能找到原像。
- **双射**：既是单射又是满射（一一对应）。

### 2. 线性空间的判定与维数

- **基本条件**：对加法和数乘**封闭**，满足八条公理。
- **常见结论**：
  - **$\mathbb{C}^n$ 在 $\mathbb{R}$ 上的维数**：$dim_{\mathbb{R}}(\mathbb{C}^n) = 2n$（每个复数需要两个实数表示）。
  - **多项式空间**：$S_A = \{f(A)\}$ 的维数等于 $A$ 的**最小多项式**的次数。
  - **可交换矩阵集**：与 $n$ 阶 Jordan 块可交换的矩阵维数为 $n$（结构为上三角且对角线元素相等）。
  - **常见子空间**：迹为 0 的矩阵、对称矩阵、上三角矩阵均构成子空间；但**行列式为 0** 的矩阵集合不构成子空间（对加法不封闭）。

### 3. 子空间的和与交

- **维数公式**：$dim(V_1 + V_2) = dim(V_1) + dim(V_2) - dim(V_1 \cap V_2)$。
- **直和判断**：若 $V_1 \cap V_2 = \{0\}$，则为直和（分解唯一）。
- **并集**：两个子空间的并集通常**不是**线性空间（除非一个包含另一个）。

------

## 二、 矩阵的秩、值域 (R) 与零空间 (N)

### 1. 维数关系

- **核心公式**：$dim(R(A)) + dim(N(A)) = n$（其中 $n$ 是 $A$ 的列数）。
- **特定空间维数**：若 $rank(A) = r$，则 $\{B \mid AB = 0\}$ 的维数为 $n(n-r)$。

### 2. 直和与分解

- **直和条件**：$rank(A) = rank(A^2)$ 是 $R(A) \oplus N(A) = \mathbb{C}^n$ 的充要条件。
- **投影矩阵 (幂等阵 $A^2=A$)**：
  - 满足 $R(I-A) = N(A)$。
  - 整个空间可以分解为 $R(A) \oplus N(A)$，但 $R(A)$ 和 $N(A)$ **不一定正交**（除非是正交投影阵）。

### 3. 方程组解的结构

- **Ax=b 有解**：当且仅当 $b \in R(A)$。
- **行空间解**：在 $R(A^T)$（行空间）中，非齐次方程 $Ax=b$ 有且仅有一个解向量。

------

## 三、 内积空间与 Hermite 矩阵

### 1. 内积的定义与性质

- **四要素**：正定性、共轭对称性、可加性、齐次性。
- **内积与基**：有限维实空间中，给定任何一组线性无关向量，都可以通过定义特定的内积使其成为标准正交基。
- **注意**：普通的线性空间不一定有内积运算，因此柯西-施瓦茨不等式 $|(x,y)| \leq ||x|| \cdot ||y||$ 只在内积空间成立。

### 2. Hermite 矩阵与迹

- **Hermite 矩阵**：满足 $A = A^H$。
- **分解**：任意复方阵 $A$ 可唯一表示为 $A = H + K$，其中 $H$ 是 Hermite 阵，$K$ 是反 Hermite 阵。
  - $H = \frac{1}{2}(A + A^H)$
  - $K = \frac{1}{2}(A - A^H)$
- **迹与内积**：对于复矩阵，$tr(AB^H)$ 定义了一个内积。
- **半正定结论**：若 $A, B$ 是 Hermite 半正定阵且 $tr(AB)=0$，则 $AB=0$。

------

## 四、 特征值与正定性

### 1. 特征值性质

- **线性变换**：同一线性变换在不同基下的矩阵是相似的，因此其**特征值完全相同**。
- **正定阵**：实对称阵正定的充要条件是特征值全部大于 0，或所有主子式大于 0。

### 2. 矩阵等价性

- $A$ 正定 $\iff$ 存在正定矩阵 $D$ 使得 $A = D^2$。
- 若 $C$ 可逆，则 $A$ 正定 $\iff C^T AC$ 正定。

------

## 五、 易错点提示 (陷阱)

1. **零向量**：在线性空间中，零向量是唯一的，但“向量 1”在特定定义下可以不是零向量。
2. **正交与无关**：**正交**向量组（不含零向量）一定线性无关；但**线性无关**向量组不一定正交（除非定义了特定内积）。
3. **维数限制**：在 3 维空间中，不可能存在一个 2 维子空间和两个 1 维子空间两两正交（总维数和 $2+1+1=4 > 3$）。
4. **Cayley-Hamilton 定理**：$A^n$ 一定可以由 $I, A, \dots, A^{n-1}$ 线性表示。

## 第二章、线性变换

## 一、 线性变换的定义与基本性质

### 1. 线性变换的判定

- **基本条件**：满足加法封闭 $T(\alpha + \beta) = T(\alpha) + T(\beta)$ 和数乘封闭 $T(k\alpha) = kT(\alpha)$。
- **典型例子**：
  - **矩阵运算**：$T(A) = A^T$ 和 $T(A) = AB$（$B$ 为固定阵）是线性变换。
  - **非线性陷阱**：$T(A) = A^*$（伴随阵）和 $T(A) = A^H$（共轭转置，因共轭不满足复数域齐次性）通常**不是**复空间上的线性变换。
  - **唯一性**：定义映射 $T(X) = tr(X)$ 是满足 $T(XY) = T(YX)$ 且 $T(I) = n$ 的**唯一**线性映射。

### 2. 变换、矩阵与基的关系

- **一一对应**：线性变换 $T$ 与方阵 $A$ 的一一对应关系**必须建立在给定一组基**的基础上。
- **坐标一致性**：若 $T$ 在**任意**一组基下的矩阵都相同，则 $T$ 必为**数乘变换**（即 $T = kI$）。
- **逆变换**：若线性变换 $T$ 可逆，其逆映射 $T^{-1}$ 必然也是线性变换。
- **复合变换**：若 $T$ 是线性变换，其幂运算 $T^m$ 也是线性变换。

------

## 二、 核心子空间：像空间 (R) 与核空间 (N)

### 1. 维数与秩

- **维数定理**：$dim(R(T)) + dim(N(T)) = n$（像空间维数 + 核空间维数 = 全空间维数）。
- **幂矩阵的秩**：对于 $n$ 阶矩阵 $A$，其秩随幂次增加而递减，但在 $n$ 步内必然稳定，即 $rank(A^n) = rank(A^{n+1})$。
- **单射判定**：若 $T$ 将一组线性无关向量映射为线性无关向量，则 $T$ 是**单射**。

### 2. 特殊矩阵的子空间性质

- **幂等阵 ($A^2 = A$)**：

  - 其像空间与核空间构成直和：$R(A) \oplus N(A) = V$。

  - 两个幂等阵 $T_1, T_2$ 像空间相等的充要条件是 $T_1T_2 = T_2$ 且 $T_2T_1 = T_1$。

  - 推导：

    ### 像空间相等 ($R(T_1) = R(T_2)$) 的充要条件

    **结论：** $R(T_1) = R(T_2) \iff T_1T_2 = T_2$ 且 $T_2T_1 = T_1$

    #### **必要性推导 ($\implies$)**

    假设 $R(T_1) = R(T_2)$：

    1. 因为 $T_2$ 是幂等阵，对任意向量 $x$，都有 $T_2x \in R(T_2)$。
    2. 由于 $R(T_2) = R(T_1)$，所以 $T_2x \in R(T_1)$。
    3. 根据幂等阵性质，若 $y \in R(T_1)$，则 $T_1y = y$。
    4. 令 $y = T_2x$，则有 $T_1(T_2x) = T_2x$。由于对任意 $x$ 均成立，故 **$T_1T_2 = T_2$**。
    5. 同理，交换 $T_1, T_2$ 角色，可得 **$T_2T_1 = T_1$**。

    #### **充分性推导 ($\impliedby$)**

    假设 $T_1T_2 = T_2$ 且 $T_2T_1 = T_1$：

    1. 由 $T_1T_2 = T_2$ 可知，$T_2$ 的每一列都是 $T_1$ 列向量的线性组合，故 $R(T_2) \subseteq R(T_1)$。
    2. 由 $T_2T_1 = T_1$ 可知，$T_1$ 的每一列都是 $T_2$ 列向量的线性组合，故 $R(T_1) \subseteq R(T_2)$。
    3. 综上所述，$R(T_1) = R(T_2)$。

- **核空间相等**：若 $T_1^2 = T_1, T_2^2 = T_2$ 且满足 $T_1T_2 = T_1, T_2T_1 = T_2$，则 $T_1$ 与 $T_2$ 拥有**相同的核空间**。

- 推导：

  ### 核空间相等 ($N(T_1) = N(T_2)$) 的充要条件

  **结论：** $N(T_1) = N(T_2) \iff T_1T_2 = T_1$ 且 $T_2T_1 = T_2$

  #### **必要性推导 ($\implies$)**

  假设 $N(T_1) = N(T_2)$：

  1. 对任意向量 $x$，考虑向量 $(I - T_2)x$。
  2. 计算 $T_2(I - T_2)x = (T_2 - T_2^2)x = 0$，说明 $(I - T_2)x \in N(T_2)$。
  3. 由于 $N(T_2) = N(T_1)$，所以 $(I - T_2)x \in N(T_1)$。
  4. 这意味着 $T_1(I - T_2)x = 0$，展开得 $T_1x - T_1T_2x = 0$，即 $T_1x = T_1T_2x$。
  5. 由于对任意 $x$ 成立，故 **$T_1T_2 = T_1$**。
  6. 同理，交换角色可得 **$T_2T_1 = T_2$**。

  #### **充分性推导 ($\impliedby$)**

  假设 $T_1T_2 = T_1$ 且 $T_2T_1 = T_2$：

  1. 若 $x \in N(T_2)$，则 $T_2x = 0$。代入 $T_1 = T_1T_2$，得 $T_1x = T_1(T_2x) = T_1(0) = 0$，故 $x \in N(T_1)$，即 $N(T_2) \subseteq N(T_1)$。
  2. 若 $x \in N(T_1)$，则 $T_1x = 0$。代入 $T_2 = T_2T_1$，得 $T_2x = T_2(T_1x) = T_2(0) = 0$，故 $x \in N(T_2)$，即 $N(T_1) \subseteq N(T_2)$。
  3. 综上所述，$N(T_1) = N(T_2)$。

- 总结：

  **“左乘谁，保谁像；右乘谁，保谁核。”**

  - 像相等：$T_1T_2 = T_2$（$T_1$ 左乘 $T_2$ 结果还是 $T_2$，说明 $T_2$ 的像被 $T_1$ 包含）。
  - 核相等：$T_1T_2 = T_1$（$T_1$ 右乘 $T_2$ 结果还是 $T_1$，说明 $T_1$ 的核受 $T_2$ 控制）。

------

## 三、 特征值与矩阵相似理论

### 1. 特征值与特征向量

- **基本不等式**：几何重数（特征子空间的维数 $dim(E(\lambda))$）永远**小于等于**其代数重数。

- **多项式性质**：若 $x$ 是 $A$ 的特征向量，则 $x$ 也是矩阵多项式 $f(A)$ 的特征向量。

- **交换性**：若 $AB = BA$，则 $A$ 与 $B$ 具有**公共的特征向量**。

- 交换性证明：

- **命题**：设 $A, B \in \mathbb{C}^{n \times n}$。若 $AB = BA$，则 $A$ 与 $B$ 至少存在一个公共的特征向量。 

  

  ### 1. 证明思路

  利用特征子空间的**不变性**。我们先取 $A$ 的一个特征子空间，证明该空间对 $B$ 是不变的，从而在该子空间内找到 $B$ 的特征向量。

  ### 2. 详细推导步骤

  第一步：取 $A$ 的特征值与特征子空间

  设 $\lambda$ 是矩阵 $A$ 的一个特征值，定义其对应的特征子空间为 $V_\lambda$：

  $$V_\lambda = \{ x \in \mathbb{C}^n \mid Ax = \lambda x, x \neq 0 \}$$

  

  显然，$V_\lambda$ 是一个非空的线性子空间。

  第二步：证明 $V_\lambda$ 是 $B$ 的不变子空间

  对于任意 $x \in V_\lambda$，我们需要判断 $Bx$ 是否仍然属于 $V_\lambda$。

  计算 $A$ 作用在 $Bx$ 上的结果：

  

  $$A(Bx) = (AB)x$$

  

  由于题目给定 $AB = BA$：

  

  $$(AB)x = (BA)x = B(Ax)$$

  

  因为 $x \in V_\lambda$，有 $Ax = \lambda x$：

  

  $$B(Ax) = B(\lambda x) = \lambda(Bx)$$

  

  由此得到 $A(Bx) = \lambda(Bx)$。这意味着 $Bx$ 满足 $A$ 的特征向量定义，即 $Bx \in V_\lambda$。

  因此，$V_\lambda$ 是 $B$ 的不变子空间。

  第三步：在不变子空间内寻找 $B$ 的特征向量

  我们将矩阵 $B$ 限制在子空间 $V_\lambda$ 上，考虑线性变换 $B|_{V_\lambda} : V_\lambda \to V_\lambda$。

  根据代数基本定理，定义在复数域上的任何线性变换至少有一个特征值。

  设 $\mu$ 是 $B|_{V_\lambda}$ 的一个特征值，则必然存在一个非零向量 $v \in V_\lambda$，使得：

  

  $$Bv = \mu v$$

  第四步：结论成立

  此时，向量 $v$ 满足：

  1. 由于 $v \in V_\lambda$，故 $Av = \lambda v$（$v$ 是 $A$ 的特征向量）。

  2. 由于 $Bv = \mu v$，故 $v$ 也是 $B$ 的特征向量。

     由此证得，$v$ 是 $A$ 与 $B$ 的公共特征向量。

**考点链接**：该结论是**“同时可对角化”**的必要条件。若 $A$ 和 $B$ 均可对角化且 $AB=BA$，则存在相同的变换矩阵 $P$ 使得 $P^{-1}AP$ 和 $P^{-1}BP$ 同时为对角阵 3。



**避坑指南**：

- **单向性**：有公共特征向量**推不出** $AB=BA$（除非所有特征向量都相同）。
- **数域要求**：在实数域 $\mathbb{R}$ 上不一定成立（可能无实特征值），但在复数域 $\mathbb{C}$ 上必然成立。

**常用结论**：如果 $A$ 的特征值均不相同（单特征值），且 $AB=BA$，那么 $B$ 的每一个特征向量必然都是 $A$ 的特征向量。

### 2. 相似与对角化

- **相似性**：属于同一线性变换在不同基下的矩阵，其特征值完全相同。
- **可对角化充要条件**：所有特征值的几何重数等于代数重数。
- **特殊对角化**：满足 $A^2 = kA$ ($k \neq 0$) 的矩阵一定可以相似对角化。（特殊情况，幂等阵）

### 3. $AB$ 与 $BA$ 的关系

- **特征值**：$AB$ 与 $BA$ 具有相同的**非零**特征值。（行列式换位公式）
- **迹**：恒有 $tr(AB) = tr(BA)$。
- **秩**：$rank(AB)$ 与 $rank(BA)$ **不一定相等**。AB=O,BA$\neq$O

------

## 四、 内积空间与特殊变换

### 1. 欧氏空间与对称变换

- **对称变换判定**：若 $T$ 在标准正交基下的矩阵是实对称阵，则满足 $(T(x_i), x_j) = (x_i, T(x_j))$。
- **度量矩阵**：设基的度量矩阵为 $G$，正交变换 $T$ 的矩阵为 $A$，则满足关系 $A^TGA = G$。

### 2. 正交变换与保性

- **定义**：正交变换是保持向量**长度（范数）不变**的线性变换。
- **判定陷阱**：
  - “保持夹角不变”是必要条件但非充分条件（数乘变换（2x）也保夹角）。
  - 正交变换将标准正交基映射为**另一组标准正交基**。

------

## 五、 重点公式与计算技巧

- **行列式计算**：
  - 若 $A$ 为秩 $r$ 的 $n$ 阶幂等阵，则 $|A + 2I| = 3^r \cdot 2^{n-r}$。
  - 若 $A$ 为幂零阵，则 $|A + I| = 1$。
- **秩的不等式**：$rank(ABC) \geq rank(AB) + rank(BC) - rank(B)$ 恒成立。
- **实矩阵特殊结论**：对于实矩阵，$rank(A) = rank(A^T A) = rank(AA^T)$。

# 三章、矩阵分解

## 一、 $\lambda$ 矩阵、Jordan 标准型与相似性

### 1. $\lambda$ 矩阵与等价判定

- **可逆性**：$n$ 阶 $\lambda$ 矩阵 $A(\lambda)$ 可逆的充要条件是其行列式 $|A(\lambda)|$ 为**非零常数**。仅秩为 $n$ 是不够的。
- **相抵与相似**：
  - 秩相等的两个长方 $\lambda$ 矩阵不一定相抵。
  - 两矩阵相似的充要条件是它们的**不变因子**相同，或其特征矩阵 $\lambda I - A$ 与 $\lambda I - B$ **等价**。
- **初等因子**：若两个矩阵具有相同的特征多项式和最小多项式，它们**不一定相似**（除非每个特征值对应的 Jordan 块只有一个）。

### 2. Jordan 标准型性质

- **转置一致性**：矩阵 $A$ 与其转置 $A^T$ 具有相同的 Jordan 标准型。
- **块的个数**：特征值 $0$ 对应的 Jordan 块个数等于其**几何重数** $d$。
- **单纯矩阵（可对角化）**：
  - 充要条件：最小多项式无重根。
  - 满足 $A^2 = kA$ ($k \neq 0$) 或 $A^n = I$ 的矩阵必为单纯矩阵。

------

## 二、 矩阵分解与特殊矩阵

### 1. 满秩分解 $A = BC$

- **性质**：若 $A$ 为 $m \times n$ 且秩为 $r$，则 $B$ 是 $m \times r$ 列满秩，$C$ 是 $r \times n$ 行满秩。
- **同解性**：$Ax=0$ 与 $Cx=0$ 是同解方程组。
- **空间关系**：像空间 $R(A) = R(B)$；零空间 $N(A) = N(C)$。
- **投影矩阵**：若 $A=A^2$ 且 $A=BC$ 为满秩分解，则 $CB = I$。

### 2. 奇异值分解 (SVD) 与谱分解

- **奇异值**：$A$ 的奇异值是 $A^H A$ 特征值的非负平方根。它们在酉变换下保持不变（酉不变性）。
- **分解存在性**：
  - **QR 分解**：适用于列满秩矩阵。
  - **谱分解**：仅适用于**正规矩阵**（如 Hermite 阵、实对称阵、酉矩阵）。
  - **奇异值分解**：适用于任何矩阵。
- **极分解**：可逆实矩阵 $A$ 一定可以表示为正交矩阵 $Q$ 和正定矩阵 $S$ 的乘积 ($A=QS$)。

------

## 三、 Hermite 矩阵与正定性

### 1. Hermite 阵与正规阵

- **酉相似对角化**：矩阵可酉相似于对角阵的充要条件是它是**正规矩阵** ($AA^H = A^H A$)。
- **Hermite 阵性质**：若 $A, B$ 均为 Hermite 阵，则它们酉相似的充要条件是**特征值相同**。
- **乘积特征值**：
  - 若 $A$ 正定，$B$ 半正定，则 $AB$ 的特征值均为**非负实数**。
  - 若 $A, B$ 均正定，则 $AB$ 的特征值均为**正实数**。
  - $A, B$ 为对称正定阵时，$A+B$ 也是对称正定阵。

### 2. 常见判定陷阱

- **反 Hermite 阵**：若 $A$ 是正定 Hermite 阵，$B$ 是反 Hermite 阵，则 $AB$ 特征值的实部为 $0$。
- **正交投影矩阵**：满足 $A^2=A$ 且 $A^H=A$。其性质包括 $N(A-I)$ 是 $N(A)$ 的正交补空间。

------

## 四、 矩阵秩与特征值的核心公式

### 1. 换位公式与特征值

- **特征值一致性**：$AB$ 与 $BA$ 具有相同的**非零**特征值。
- **迹恒等**：$tr(AB) = tr(BA)$。
- **秩的关系**：$rank(AB)$ 与 $rank(BA)$ **不一定相等**。但 $rank(A) = rank(A^H A) = rank(AA^H)$ 恒成立。

### 2. 伴随矩阵 $A^*$ 的秩

- 若 $rank(A) = n$，则 $rank(A^*) = n$。
- 若 $rank(A) = n-1$，则 $rank(A^*) = 1$。
- 若 $rank(A) < n-1$，则 $rank(A^*) = 0$。

## 五、 矩阵计算技巧与小结论

- **谱半径 $\rho(A)$**：是矩阵特征值模的最大值。若 $A^3 + A^2 - A - E = 0$，则特征值只能是 $1$ 或 $-1$，谱半径为 $1$。
- **Householder 矩阵**：属于正交阵且是对合阵 ($H^2=I$)，一定可以进行谱分解。
- **同解方程组**：$A^T Ax = 0$ 与 $Ax = 0$ 永远是同解方程组。
- **公共特征向量**：若 $AB=BA$，则 $A$ 和 $B$ 至少有一个公共特征向量。

# 四章、矩阵范数与矩阵函数

### 一、 向量范数与矩阵范数的基本性质

#### 1. 向量范数判定与不等式

- **$p$-范数条件**：当 $p \ge 1$ 时满足三角不等式；当 $0 < p < 1$ 时不满足三角不等式，不构成范数。
- **范数判定陷阱**：定义 $\|x\| = \sqrt{x^T Ax}$ 要成为范数，矩阵 $A$ 必须是**正定**矩阵，若仅为实对称则缺少正定性（不能保证非负性及正性）。
- **范数等价性定理**：在有限维空间中，所有范数都等价。例如：
  - $\|x\|_2 \le \|x\|_1 \le \sqrt{n}\|x\|_2$。
  - $\|x\|_\infty \le \|x\|_1 \le n\|x\|_\infty$。

#### 2. 矩阵范数的分类与性质

- **算子范数（诱导范数）**：
  - **谱范数 ($\|A\|_2$)**：等于 $A^H A$ 最大特征值的平方根（最大奇异值 $\sigma_1$）。
  - **谱范数等价定义**：$\|A\|_2 = \max_{\|x\|_2=\|y\|_2=1} |y^H A x|$。
  - **重要不等式**：$\|A\|_2^2 \le \|A\|_1 \|A\|_\infty$；$\|A^H A\|_2 = \|A\|_2^2$。
- **Frobenius 范数 ($\|A\|_F$)**：
  - 定义为矩阵元素平方和的平方根，也等于**奇异值平方和**的平方根。
  - **Schur 估计**：$\|A\|_F^2 \ge \sum |\lambda_i|^2$，当且仅当 $A$ 为**正规阵**时取等号。
  - **秩 1 判别**：若 $\|A\|_F = \|A\|_2$，则矩阵 $A$ 的秩为 1。
  - **注意**：$\|A\|_F$ **不是**算子范数。

#### 3. 相容性 (Compatibility)

- **定义**：满足 $\|Ax\|_V \le \|A\|_M \cdot \|x\|_V$。
- **结论**：对于任意矩阵范数，一定存在至少一种向量范数与其相容。
- **不可逆阵结论**：若 $A$ 不可逆，则对于任何相容范数，恒有 $\|I - A\| \ge 1$。

------

### 二、 矩阵序列与矩阵函数的收敛性

#### 1. 幂序列 $A^k$ 的收敛性

- **收敛到零矩阵**：充要条件是谱半径 $\rho(A) < 1$。
- **收敛到非零矩阵**：
  - 若 $A^k$ 收敛到 $B$，则 $A$ 必须满足：谱半径 $\rho(A) \le 1$；且特征值为 1 的 Jordan 块均为 1 阶；特征值模为 1 且不为 1 的特征点不存在。
  - 若 $A^k$ 收敛到自身，则 $A$ 是**幂等阵** ($A^2 = A$)。
- **谱半径与范数关系**：对于任意范数，谱半径 $\rho(A) \le \|A\|$。特别地，若 $A$ 是正规阵，则 $\rho(A) = \|A\|_2$。
- 谱半径取1时，最好矩阵级数的每个元素都检查是否收敛，

#### 2. 矩阵指数函数 $e^A$

- **性质**：
  - 转置关系：$(e^A)^T = e^{A^T}$。
  - **反对称阵性质**：若 $A$ 是反对称阵，则 $e^A$ 是正交矩阵。

### 三、 特征值估计与摄动理论

#### 1. 盖尔圆定理 (Gershgorin Circle Theorem)

- 用于估计特征值的范围。若所有盖尔圆都位于复平面实轴上或通过盖尔圆分布判别，可推断特征值是否均为实数。

#### 2. 矩阵摄动与巴拿赫 (Banach) 引理

- **可逆性判别**：若 $\|A^{-1}(B - A)\| < 1$，则 $B$ 必可逆。

- **推导过程**：

  1. 构造关联形式：我们可以将矩阵 $B$ 改写为包含 $A$ 的形式：

     

     $$B = A + (B - A) = A[I + A^{-1}(B - A)]$$

  2. 利用 Neumann 级数定理：

     矩阵理论中有一个重要引理：若 $\|M\| < 1$，则 $(I + M)$ 必可逆，且其逆矩阵可以用级数表示为 $(I + M)^{-1} = \sum_{k=0}^{\infty} (-M)^k$。

  3. 应用引理：

     令 $M = A^{-1}(B - A)$。根据已知条件 $\|M\| < 1$，因此 $[I + A^{-1}(B - A)]$ 是可逆的。

  4. 结论：

     由于 $B$ 是两个可逆矩阵 $A$ 与 $[I + A^{-1}(B - A)]$ 的乘积，根据可逆矩阵的性质（乘积仍可逆），$B$ 必可逆。

- **误差估计公式**：

  - $\|B^{-1}\| \le \frac{1}{a - b}$（其中 $a = 1/\|A^{-1}\|$, $b = \|B-A\|$）。
  - $\|B^{-1} - A^{-1}\| \le \|B^{-1}\| \cdot \|A - B\| \cdot \|A^{-1}\|$。

- 推导过程：

- **命题**：$\|B^{-1}\| \le \frac{1}{a - b}$，其中 $a = 1/\|A^{-1}\|, b = \|B - A\|$。

  **推导过程**：

  1. **利用逆矩阵表达式**：由上述推导知 $B^{-1} = [I + A^{-1}(B - A)]^{-1} A^{-1}$。

  2. 取范数并利用次可乘性：

     

     $$\|B^{-1}\| \le \|[I + A^{-1}(B - A)]^{-1}\| \cdot \|A^{-1}\|$$

  3. 利用算子不等式：对于可逆矩阵 $(I+M)$，其逆的范数满足 $\|(I+M)^{-1}\| \le \frac{1}{1 - \|M\|}$。

     这里 $\|M\| = \|A^{-1}(B-A)\| \le \|A^{-1}\| \cdot \|B-A\|$。

  4. 代入变量：

     

     $$\|B^{-1}\| \le \frac{\|A^{-1}\|}{1 - \|A^{-1}\| \cdot \|B-A\|}$$

  5. 分子分母同除以 $\|A^{-1}\|$：

     

     $$\|B^{-1}\| \le \frac{1}{\frac{1}{\|A^{-1}\|} - \|B - A\|} = \frac{1}{a - b}$$

     

     推导完成。

  ------

  ### 3. 误差估计公式推导

  **命题**：$\|B^{-1} - A^{-1}\| \le \|B^{-1}\| \cdot \|A - B\| \cdot \|A^{-1}\|$。

  **推导过程**：

  1. 恒等式变形：考虑差值矩阵 $B^{-1} - A^{-1}$，利用“左乘 $B^{-1}B$ 右乘 $AA^{-1}$”的技巧：

     

     $$B^{-1} - A^{-1} = B^{-1}(I) - (I)A^{-1} = B^{-1}(AA^{-1}) - (B^{-1}B)A^{-1}$$

     $$= B^{-1}(A - B)A^{-1}$$

  2. 取范数：利用范数的次可乘性（$\|XYZ\| \le \|X\| \cdot \|Y\| \cdot \|Z\|$）：

     

     $$\|B^{-1} - A^{-1}\| = \|B^{-1}(A - B)A^{-1}\| \le \|B^{-1}\| \cdot \|A - B\| \cdot \|A^{-1}\|$$

     

     推导完成。

### 四、 特殊矩阵的补充结论

- **正规阵 ($AA^H = A^H A$)**：乘积不一定是正规阵，除非它们可交换。
- **循环阵**：特征根的模小于等于行的模和；循环阵的和与乘积仍为循环阵。
- **酉不变性**：谱范数和 Frobenius 范数具有酉不变性，即乘一个酉矩阵后，奇异值不变，范数也不变。
- **Householder 矩阵**：既是正交矩阵又是投影矩阵。
